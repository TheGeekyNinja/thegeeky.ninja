Recruitment process
 - Define Position
 - Ask for referrals internally
 - Create job post on appropriate job site
 - Add applicants to Planka recruitment board
 - Basic screening: Resume, experience
 - Text basic interview
 - Management review
 - Voice/video interview (if applicable)
 - Offer/rejection
 - Onboarding
 - Trial
 - Review
